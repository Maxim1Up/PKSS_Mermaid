      Кассир открывает раздел "Билеты": 5: Пользователь

    section Аналитика и отчеты
      Менеджер заходит в раздел "Аналитика": 5: Пользователь
      Система отображает статистику по продажам и посещаемости: 5: Система
      Менеджер просматривает отчеты о популярных фильмах: 4: Пользователь
      Система обновляет данные в реальном времени: 5: Система
    section Уведомления и завершение
      Система отправляет уведомления о новых фильмах и акциях: 4: Система

      Пользователь завершает работу с системой и выходит: 5: Пользователь
